Syllabus

The course is focused on learning how to use CSS, HTML and JS, to build web applications without the help of frameworks or libraries. We are going to use two books from SitePoint for the course, and the technologies we’ve been using so far (Git, GitHub, VS Code).

The first 4.5 weeks we will work on a team project following the course. Weeks 5-7 we will work on our own team project. The last 4.5 weeks all class will work on something that it is not pretty clear to me.

Our final grade will be based of our own raking against the course outcomes at the end of the semester, and a portfolio that we should start creating along the semester.

Working with groups

The main issue with groups is lack of communication, from which other issues arise. One of these issues is not finding a time to work together. We should prioritize the time to meet with our teams, but we should look a way to work asynchronously as much as we can.

There are some tools to help us achieve this goal of working asynchronously like Slack (Or MS Teams), GitHub and Trello.
